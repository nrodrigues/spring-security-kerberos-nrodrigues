Login with Username and Password

  • With "spnego-with-form-login.xml" active, you can enter any username and password is always "notUsed".
  • With "spnego-with-server-side-kerberos-option.xml" or "server-side-kerberos.xml" active, you can enter any Kerberos user with the corresponding password.
  • The login doesn't work with "spnego.xml" active
You can activate the different configurations in web.xml.
